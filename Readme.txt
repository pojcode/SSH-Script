===============================================================================
| Script created to perform ssh session to multiple devices at the same time. |
| 	- Multi-vendor							      |
|	- Support external config files for devices			      |
|	- Support diferent configs for asynchronous pings and ssh sessions    |
|	- A log.txt will be created after script ends			      |
|	- More features...						      |
===============================================================================

LINUX, MAC users:
	You must need Python v3.12.2 or onwards to run script
	Lower Python versions could run script but not tested

	Download Python here or use your packet manager, up to you
	https://www.python.org/

	You will need Python module netmiko from Pypi repository
		run in shell -> pip install netmiko
